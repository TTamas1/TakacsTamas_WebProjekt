Takács Tamás

https://ttamas1.github.io/TakacsTamas_WebProjekt/

A weboldal eladó/kiadó ingatlanok nyilvántartásával foglalkozik. Lehet törölni, hozzáadni, keresni, illetve bejelentkezni is.

A JavaScript kód a "JS" nevű mappában található. 
Feladás.js nevű fájl, amelynek segitségvel ingatlan hirdetéseket adhatunk fel. 
index.js fájl, ami az ingatlanokat tartalmazza, a főoldal más néven.
kereses.js fájl, amelynek segitségével kilistázhatjuk magunknak a számunkra releváns hirdetéseket
login.js fájl, ami lehetőve teszi a bejelentkezést, ugyan nincs megoldva a felhasználó kezelés.
torol.js fájl, amivel azonosito alapján törölhetjük az ingatlan hirdetéseket.

A CSS nevű mappában, a style.css nevű fájlban található meg a google fonts amit használtam az oldal készitéséhez.

Az oldal elkészitéséhez korábbi munkáimat, az órai anyagokat, és a ChatGPT, illetve Google Gemini AI-okat használtam.
